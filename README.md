#MongoDB Signup & Login System (PHP)

This project demonstrates how to integrate **MongoDB Community Server** with a **PHP web application** to implement a complete **Signup and Login system**.  
It includes full **CRUD operations**, validation, and proper project organization similar to an industry-level backend project.

---

## 📌 Features

- User Signup using MongoDB
- User Login authentication
- Password hashing for security
- Dashboard after successful login
- Update user details (Update operation)
- Delete user account (Delete operation)
- Prevent duplicate email/username
- Proper validation and error handling
- Secure configuration using `.env` file

---

## 🛠️ Technologies Used

- PHP 8+
- MongoDB Community Server
- MongoDB PHP Driver
- Composer
- XAMPP (Apache Server)
- VS Code
- Git & GitHub
